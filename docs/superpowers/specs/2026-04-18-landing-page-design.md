# WhatStage Landing Page — Design Spec

## Overview

A Shopify-grade SaaS marketing landing page to convert businesses into WhatStage tenants. Light theme inversion of the DESIGN.md dark palette. Center-aligned "Centered & Ethereal" layout with scroll-triggered entrances and ambient micro-animations.

The bar: this must look like it was designed by a well-funded startup's in-house design team, not generated by AI. Every spacing value, shadow layer, and typographic weight is deliberate.

## Route

`src/app/(marketing)/page.tsx` — replaces the current placeholder hero.

### Atmospheric Touches

- Hero section: soft radial gradient `radial-gradient(ellipse at 50% 0%, rgba(54,244,164,0.06) 0%, transparent 60%)` — barely visible green wash from top center
- Alternate sections: `#FAFBFC` background, not pure white — creates subtle depth without being obvious
- No harsh color transitions between sections — use generous padding to breathe

## Typography

Exactly per DESIGN.md. Since we don't have NeueHaasGrotesk licensed, use the Geist font family already installed (similar refined sans-serif). Geist supports weights 100-900 but not 330 — use weight 300 (light) for display text as the closest match to 330's ethereal feel. Apply `font-feature-settings: 'ss03'` throughout.

## Animations Implementation

All animations use pure CSS + Intersection Observer. No external libraries.

### Scroll-triggered Entrances

```
Initial state: opacity 0, transform translateY(20px)
In-view state: opacity 1, transform translateY(0)
Transition: 600ms cubic-bezier(0.16, 1, 0.3, 1)
Stagger: 150ms delay between sibling elements
Trigger: IntersectionObserver with threshold 0.1
```

A single `useScrollReveal` hook handles all scroll animations. Elements get a `data-reveal` attribute and animate when they enter the viewport.

### Ambient Animations

```css
/* Floating cards in hero */
@keyframes float {
  0%, 100% { transform: translateY(0px) }
  50% { transform: translateY(-10px) }
}
/* Apply with staggered delays: 0s, 1.5s, 3s, 4.5s */
/* Duration: 4-6s each, ease-in-out, infinite */

/* Subtle CTA pulse glow */
@keyframes pulse-glow {
  0%, 100% { box-shadow: 0 0 0 0 rgba(54, 244, 164, 0) }
  50% { box-shadow: 0 0 20px 4px rgba(54, 244, 164, 0.15) }
}

/* Navbar background transition handled by scroll state class toggle */
```

### Micro-interactions

- Button hover: `transform: scale(1.02)`, shadow layer expands, 200ms ease
- Card hover: `transform: translateY(-2px)`, shadow deepens, 200ms ease
- Nav link hover: color fade, 200ms ease
- FAQ chevron: `transform: rotate(180deg)`, 300ms ease

## Quality Checklist

- [ ] Every shadow is multi-layered (no single `box-shadow` values)
- [ ] Display text uses weight 330-400, never heavier
- [ ] Neon Green appears only on: overline badge bg, pricing badge, focus rings — nowhere else as a large surface
- [ ] Pill buttons use 9999px radius with asymmetric padding
- [ ] Section spacing is 120px (theatrical, not cramped)
- [ ] `font-feature-settings: 'ss03'` on all text elements
- [ ] Zinc-based neutral scale for text hierarchy (not random grays)
- [ ] Cards use inset white glow + stacked shadow approach
- [ ] Footer is the only dark section (contrast anchor)
- [ ] No generic stock photo vibes — all visual interest comes from typography, spacing, and subtle animation
