How Metrics Are Computed

Phase success rate

entered = COUNT(conversation_phases
  WHERE phase_id = X)

advanced = COUNT WHERE exit_reason
  IN ('advanced', 'converted')

dropped = COUNT WHERE exit_reason
  = 'dropped'

success_rate = advanced / entered

Drop-off detection (background job)

Every 15min: scan open phases
WHERE entered_at < now() - delay
AND exited_at IS NULL
AND no messages since entered_at

→ send follow_up_message via FB
→ if still no reply after delay*2:
  set exit_reason = 'dropped'

Conversion tracking

On lead_event INSERT:
type = 'appointment_booked' |
       'purchase' | 'form_submit'

→ match against campaign.goal
→ INSERT campaign_conversions
→ mark conversation_phase
  exit_reason = 'converted'

Experiment suggestion logic

For each campaign in experiment:
conv_rate = conversions / assigned

suggest_winner IF:
· all variants >= min_sample_size
· best_rate > others by >10%

Show banner: "Variant X is winning
— promote when ready"

Drop-off timing — two-step

1. After follow_up_delay_minutes of silence → send follow-up message
2. After another follow_up_delay_minutes with still no reply → mark exit_reason = 'dropped' and close phase

This means "dropped" means the lead ignored even the follow-up — a true cold lead.
