Design Section 4: Complete Feature Map

Every feature WhatStage needs, organized by subsystem — this is what the bootstrap skill will generate

🔐 Auth & Multi-tenancy

Auth Flow Supabase auth — signup, login, callback, session management
Tenant Routing Wildcard subdomain resolution — middleware extracts tenant slug, resolves tenant_id
Tenant Management Settings page — business type, bot goal, FB credentials, team members
Onboarding New tenant creation wizard — slug, name, business type, bot goal, FB page connect

💬 Messenger Bot Engine

Webhook Handler Receive & verify FB webhooks — signature validation, event parsing, tenant routing
Message Handling Process incoming messages — create/update lead, log conversation, trigger bot flows
Send API Send messages back — text, images, action buttons with URLs to action pages
Bot Flows Configurable triggers — keyword match, first message, postback → response templates
AI Reasoning HuggingFace integration — context evaluation, intent detection, stage recommendation

📊 Lead Management

Lead Pipeline Kanban board — drag leads between stages, filter by tags, search
Lead Profile Detail view — FB info, stage history, activity timeline, tags, notes
Activity Tracking Event log — messages, form submits, appointments, purchases, stage changes
Stage Management Configure stages — add/remove/reorder, custom colors, auto-move rules

🎯 Action Pages

Form Pages Configurable lead capture forms — fields, validation, submit → lead_event
Calendar Booking Appointment scheduler — available slots, booking confirmation, reminders
Sales Pages Landing pages — product showcase, CTA buttons, testimonials
Product Catalog E-commerce pages — product grid, detail view, cart, add to cart
Checkout Order completion — cart summary, payment reference, order confirmation
Action Page Builder Dashboard UI — create/edit action pages, configure fields, preview, publish

🛒 Commerce

Product Management CRUD products — name, price, images (Cloudinary), stock tracking
Order Management View/manage orders — status updates (pending → paid → fulfilled), order history
Appointment Management View/manage bookings — confirm, cancel, reschedule, calendar view

⚡ Workflows & Automation

Workflow Engine Core execution — trigger evaluation, step processing, run logging
Workflow Builder Dashboard UI — visual step editor, trigger config, enable/disable
Workflow Steps Step types — send message, send image, wait, condition, move stage, tag, HTTP call
Workflow Triggers Event-based — on form submit, on purchase, on stage change, on appointment booked

📱 Tenant Dashboard

Dashboard Home Overview — lead count, recent activity, stage distribution, quick actions
Conversation Inbox Live chat view — message history per lead, send manual messages, real-time via Supabase
Analytics Funnel metrics — conversion rates, action page performance, lead sources

Total: 28 Features across 7 Subsystems

Combined with 16 DB entities, ~15 UI components/pages, and ~8 core user flows, the bootstrap skill will generate approximately 70+ interconnected Obsidian notes.
